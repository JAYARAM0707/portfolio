# Portfolio Website

My personal full-stack portfolio website built to showcase my work, skills, and experience.

## 🛠️ Tech Stack

**Frontend**
- React.js + Vite
- Tailwind CSS
- React Router
- Framer Motion
- Lucide Icons

**Backend**
- Python + FastAPI
- MongoDB (Motor async driver)
- OpenAI GPT-4 API

**Deployment**
- Frontend → Vercel
- Backend → Render
- Database → MongoDB Atlas

## 📁 Project Structure

​```
portfolio/
├── frontend/     # React + Vite app
└── backend/      # Python FastAPI server (coming soon)
​```

## 🚀 Getting Started

### Frontend

​```bash
cd frontend
npm install
npm run dev
​```
